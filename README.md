# Data Structures Projects
- Bu repo, data structures pratiklerimi göstermek amacıyla oluşturuldu.
- İçerisindeki 4 farklı proje bulunmakta.
  
------------

### 2D Cell Grid

- Dinamik olarak değiştirilebilen ve elemanlarına head node aracılığıyla erişilebilen 2 boyutlu bir grid data structure'ı oluşturuldu. Her node'dan 4 farklı yöne giderek elemanlara ulaşılabilir ve kodun öne çıkan yönü az önce bahsedildiği gibi dinamik değişimlere uyum sağlamasıdır.

------------

### BST Market Items Sort

- Bir marketteki eşyaların fiyat veya son kullanma tarihine göre dizilip en optimal şekilde aranan kriterlere göre ürüne ulaşılabilmesi için bir Binary Search Tree tasarlandı.

------------

### AVL Tree Find Vezier

- Pargalı İbrahim Paşa'nın Sultan Süleyman ile sohbetinden yola çıkarak hazırlanan bu projede, maaşların karşılaştırılması için kurulan AVL Tree ile seçilen bir vezirden önce ve sonra gelen diğer vezirler bulunur, ağaç dinamik olarak update edilir.

------------

### PageRank Algorithm

- Bu proje Larry Page ve Sergey Brin'in Google'ı kurarken oluşturdukları PageRank Algorithm'in bir implementasyonudur. Graph yapıları ve Breadth-First Search üzerine yoğunlaşılmaktadır.
